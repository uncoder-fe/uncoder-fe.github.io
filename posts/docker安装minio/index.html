docker安装minio

By uncoder-fe

March 15, 2022

使用docker安装minio/minio镜像

sudo docker pull minio/minio

sudo docker run \
  -d \
  -p 9000:9000 \
  -p 9001:9001 \
  --name minio1 \
  -v ~/minio/data:/data \
  -e "MINIO_ROOT_USER=minio" \
  -e "MINIO_ROOT_PASSWORD=minio@123" \
  minio/minio server /data --console-address ":9001"

宿主数据目录: ~/minio/data 用户名: minio 密码: minio@123

访问

http://192.168.x.x:9001/login
