常见排序算法-插入排序

By uncoder

November 5, 2016

插入排序，从第二个元素挨个和前面所有元素比较,大的后移

写法

function cha(arry){
    var length = arry.length;
    for(var i=1;i<length;i++){
        var switchValue = arry[i];
        for(var j=i-1;j>=0 && arry[j]>switchValue;j--){
            arry[j+1] = arry[j];
        }
        arry[j+1]= switchValue;
    }
    console.log("结果",arry);
}
